# Instructions
这是一个用于计算某个员工在一段时间内的上班工时的一个web页面工具。

## 基本思路
操作步骤：

1. 用户打开页面，填写午休时长，默认为1.5小时。
2. 用户上传自己的打卡记录截图，调用大模型来识别截图中的打卡时间，注意，一天里面可能会有多次打卡时间，你要取最早的时间作为上班时间，最晚的时间作为下班时间。
3. 根据午休时长和上班下班时间，计算出当天的出勤时长，公式为：出勤时长 = 下班时间 - 上班时间 - 午休时长，单位为 小时，结果保留两位小数。
4. 得出每天的出勤时长，用户可以在页面中可以对自己的出勤时长进行分析，比如计算平均时长，可以选择柱状图或者折线图展示。

## 技术选型要求

- 使用最新的版本
- 启动命令使用makefile来制作
